# P2P File Transfer 🔄

A **peer-to-peer file transfer web app** built with **WebRTC** and **Socket.IO**.  
Enables users to send files directly between browsers — no central storage required.  

## 🚀 Key Highlights
- Real-time **room-based signaling** with Socket.IO  
- **Direct browser-to-browser** transfer using WebRTC DataChannels  
- Progress indicator + simple UI  
- Secure — server only used for signaling, not file storage  

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js, Express, Socket.IO  
- Web APIs: WebRTC (RTCPeerConnection, DataChannel)
